# PhotoBook

개인용 A4 포토북 편집기입니다. 사진을 페이지별 레이아웃에 배치하고, 캡션과 표지 정보를 입력한 뒤 전체 페이지를 PNG 또는 JPEG로 저장할 수 있습니다.

## 실행

```bash
npm install
npm run dev
```

프로덕션 빌드는 다음 명령으로 확인합니다.

```bash
npm run build
```

코드 품질 검사는 다음 명령으로 실행합니다.

```bash
npm run lint
```

## 주요 기능

- 표지, 1장, 2장, 3장, 4장, 5장, 6장 페이지 레이아웃
- 클릭 업로드와 드래그 앤 드롭 사진 추가
- 사진별 캡션 입력
- 사진 맞춤 방식 전환: 꽉 채우기 / 원본 비율
- 페이지 추가, 삭제, 앞뒤 이동
- 전체 배경색 변경
- 브라우저 로컬 저장소 자동 저장
- 전체 페이지 PNG/JPEG 저장

## 참고

- 사진 데이터는 브라우저의 로컬 저장소에 저장됩니다.
- 다른 브라우저나 다른 기기에서는 저장된 프로젝트가 자동으로 공유되지 않습니다.
- 이미지 크기 제한이나 압축 기능은 개인 사용 목적에 맞춰 넣지 않았습니다.
