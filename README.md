# PhotoBook

개인용 A4 포토북 편집기입니다. 사진을 페이지별 레이아웃에 배치하고, 캡션과 텍스트, 스탬프를 더한 뒤 전체 페이지를 PNG 또는 JPEG로 저장할 수 있습니다.

## 실행

```bash
npm install
npm run dev
```

프로덕션 빌드는 다음 명령으로 확인합니다.

```bash
npm run build
```

코드 품질 검사는 다음 명령으로 실행합니다.

```bash
npm run lint
```

## 주요 기능

- 표지, 1장, 2장, 3장, 4장, 5장, 6장 페이지 레이아웃
- 가져온 A4 레이아웃과 양면 펼침면 레이아웃
- 클릭 업로드와 드래그 앤 드롭 사진 추가
- 사진별 캡션 입력
- 레이아웃 텍스트 입력과 폰트, 언어, 글자 크기 설정
- 사진 맞춤 방식 전환: 꽉 채우기 / 원본 비율
- 스탬프 드래그 배치
- 본문 페이지 수 선택, 페이지 순서 변경, 전체 페이지 보기
- 전체 배경색 변경
- 인쇄 보증 외 가이드 표시 전환
- 브라우저 로컬 저장소 자동 저장
- 실행 취소 / 다시 실행
- 작업 파일(`.photobook`) 저장과 불러오기
- 전체 페이지 PNG/JPEG 저장
- 개별 파일 다운로드 또는 ZIP 묶음 저장

## 참고

- 사진 데이터는 브라우저의 로컬 저장소에 저장됩니다.
- 다른 브라우저나 다른 기기에서는 저장된 프로젝트가 자동으로 공유되지 않습니다.
- 다른 기기로 옮길 때는 작업 파일로 저장한 뒤 해당 파일을 불러오세요.
- 사진 업로드 크기 제한은 앱 설정에서 조정할 수 있습니다.
